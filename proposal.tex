\textoResumo{
% TODO reescrever resumo
Classic cryptography algorithms like RSA and ECDSA base their security upon
number theory problems like factorization and discrete logarithm. Since
polynomial time quantum algorithms for solving these problems already exist,
classic algorithms are threatened and it is a good idea to develop quantum
resistant, namely, post-quantum, cryptosystems. This work aims to present a
systematic review of the post-quantum cryptosystems class called, multivariate
polynomial public key cryptography. For digital signature schemes based on
these concepts, the security is based on the problem of solving systems of
multivariate polynomial equations, which is proven to be NP-Hard, and the
isomorphism of polynomials problem. These problems are neither solvable, in
polynomial time, by classic computers nor quantum computers, hence the
aforementioned schemes are good candidates for use in post-quantum
cryptography.
}
